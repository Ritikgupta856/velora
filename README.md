# Velora

AI-powered travel planner built with Next.js, Gemini AI, and modern web technologies.

Velora helps users generate beautiful, personalized travel itineraries with:

* AI trip planning
* budget optimization
* place discovery
* maps integration
* itinerary management
* collaborative travel planning (future)

---

# Features

## MVP

* AI itinerary generation
* destination-based trip planning
* budget-aware recommendations
* beautiful itinerary UI
* save trips
* responsive experience
* maps integration

---

## Frontend

# Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/velora.git
cd velora
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```

---

# Environment Variables

Create:

```txt
.env
```

Add:

```env
NEXT_PUBLIC_GOOGLE_MAPS_KEY=
```
